## Prerequisites 

Node.js  
NPM  
Git
## Installation

1. move to the directory you would like to develope from
2. clone from repo
3. install dependencies
4. run locally on port 3000

```bash
  cd <projects directory>
  git clone <repo link>
  cd solar_system
  npm install
  npm run start
```
